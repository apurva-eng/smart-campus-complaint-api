# Smart Campus Complaint Management API

## Description
Backend API for managing campus complaints.

## Features
- Register complaint
- View complaints
- Update status
- Delete complaint
- Authentication
- Authorization
- Validation

## Technologies
- Node.js
- Express.js

## Run

npm install

npm start

## API Endpoints

GET /complaints

GET /complaints/:id

POST /complaints

PUT /complaints/:id

DELETE /complaints/:id

POST /login
